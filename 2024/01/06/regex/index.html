正则表达式笔记

Created2024-01-06Updated2024-01-07

正则表达式笔记

Donald Lee2024-01-062024-01-07

. : 可以匹配任意单个字符, 字母, 数字甚至是 . 字符本身

绝大多数的正则表达式实现里, . 不能匹配换行符

[] : 字符集合,出现在[和]之间的所有字符都是该集合的组成部分(各个字符之间是OR的关系,而不是AND的关系)
- : 连字符定义字符区间
在一个字符集合里可以给出多个字符区间

^ : 排除某个字符集合(排除字符集合里指定的那些字符)

匹配空白的元字符
[\b] : 回退(并删除)一个字符(Backspace键)
\f : 换页符
\n : 换行符
\r : 回车符
\t : 制表符
\v : 垂直制表符

数字元字符
\d : 任何一个数字字符(等价于[0-9])
\D : 任何一个非数字字符(等价于[^0-9])

字母数字元字符
\w : 任何一个字母数字字符(大小写不敏感)或下划线字符(等价于[a-zA-Z0-9_])
\W : 任何一个非字母数字或下划线字符(等价于[^a-zA-Z0-9_])

空白字符元字符
\s : 任何一个空白字符(等价于[\f\n\r\t\v])
\S : 任何一个非空白字符(等价于[^\f\n\r\t\v])

用来匹配退格字符的[\b]不在\s的覆盖范围内, \S也没有将它排除

\x : 匹配十六进制
\0 : 匹配八进制

POSIX字符类
[:alnum:] : 任何一个字母或数字(等价于[a-zA-Z0-9])
[:alpha:] : 任何一个字母(等价于[a-zA-Z])
[:blank:] : 空格或制表符(等价于[\t ])
[:cntrl:] : ASCII控制字符(ASCII 0到31, 再加上ASCII 127)
[:digit:] : 任何一个数字(等价于[0-9])
[:graph:] : 和[:print:]一样, 但不包含空格
[:lower:] : 任何一个小写字母(等价于[a-z])
[:print:] : 任何一个可打印字符
[:punct:] : 既不属于[:alnum:], 也不属于[:cntrl:]的任何一个字符
[:space:] : 任何一个空白字符, 包括空格(等价于[\f\n\r\t\v ])
[:upper:] : 任何一个大写字母(等价于[A-Z])
[:xdigit:] : 任何一个十六进制数字(等价于[a-fA-F0-9])

POSIX模式必须以 [[ 开头, 以 ]] 结尾. 字符类必须出现在 [: 和 :] 之间

+ : 至少匹配一次一个或多个字符集合
* : 可以匹配多次,也可以一次都不匹配字符集合

? : 匹配可选文本(只能匹配某个字符或字符集合)的零次或一次出现,最多不超过一次

{} : 指定重复范围
{3} : 具体匹配次数
{2,4} : 设置区间范围, 2: 最小次数, 4: 最大次数
{3,} : 至少重复3次或更多次

贪婪型量词: *, +, {n,} (尽可能多地匹配)
懒惰型量词: ***?**, +?, {n,}? (尽可能少地匹配)

字符边界
\b : 单词边界. 用来匹配一个单词的开头或结尾
\B : 不匹配单词边界

字符串边界
^ : 代表字符串开头
$ : 代表字符串结尾(匹配换行符之后的结束位置)

(?m) : 启用多行模式(在使用时,必须出现在整个模式的最前面)

() : 子表达式, 将表达式视为单一的实体

反向引用: 指的是实体引用的是先前表达式的子表达式
\1 : 匹配模式中所使用的低一个子表达式, 以此类推

反向引用只能用来引用括号里的子表达式

大小写转换
\E : 结束\L或\U转换
\l : 把下一个字符转换为小写
\L : 把\L到\E之间的字符全部转换为小写
\u : 把下一个字符转换为大写
\U : 把\U到\E之间的字符全部转换为大写

?= : 向前查看. 指定一个必须匹配但不用在结果中返回的模式(需要匹配的文本跟在 = 的后面)
?<= : 向后查看. 查看出现在已匹配文本之前的内容

向前查看和向后查看的返回结果永远都是零长度字符串. 因此, 这两个操作有时也被称为零宽度匹配操作

向前查看模式的长度是可变的, 其中那个可以包含.和+等量词. 向后查看模式则只能是固定长度

否定式向前查看 : 向前查看不匹配指定模式的文本
否定式向后查看 : 向后查看不匹配指定模式的文本

各种环视操作
?= : 肯定式向前查看
?! : 否定式向前查看
?<= : 肯定式向后查看
?<! : 否定式向后查看

反向引用条件语法: (?(backreference)true), ? 表明这是一个条件, 括号里的backreference是一个反向引用, 仅当反向引用立即出现时, 才对表达式求值 (只在给定条件得到满足时才执行表达式)

(?(backreference)true|false) : 此语法接受一个条件和两个分别在符合/不符合该条件时执行的表达式
