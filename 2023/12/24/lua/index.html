在C++中加载Lua的一个简单示例

Lua的基本类型: nil, boolean, number, string, userdata, function, thread, table

nil: 表示非值(与C++中的nullptr类似)
boolean: 与C++中bool的作用相同
number: 涵盖了C++的int, float和double及其变体(例如long)

算术运算符: +, -, *, /, %, //
Lua在算术运算符上没有自增和自减, lua的 // 返回除法后结果的整数部分

关系运算符: <, >, <=, >=, ==, =
**=**: 测试是否相等. 与C++中的 != 运算符相同

字符串在Lua中始终是一个常量

..: 连接两个字符串

#: 检查字符串的长度

[[ 和 ]] : 分隔多行字符串

上述代码相当于 Hello\nC++\n

长字符串可以使字符串更具可读性

如果需要在长字符串内容中包含 [[ 或 ]]

等号的数量通常一个就足够了

表是Lua中唯一构建复杂数据结构的唯一方式, 它类似于C++的map容器,但更灵活

表的索引以1开始,而不是0

local : 将变量声明为块的本地变量. 局部变量有助于防止全局环境污染

Lua的控制结构

在Lua中, 不需要使用大括号和测试条件两边的括号. 需要使用 then 和 end 关键字来分隔代码分支, else 分支是可选的

在Lua中, 没有switch控制结构, 需要switch的时候可以使用 elseif 进行模拟

for使用 pairs 和 ipairs 返回表的键值对

pairs : 以未定义的顺序返回
ipairs : 按排序顺序返回

使用 pairs 遍历整个数组, 而不参考数组长度. 作为迭代器进行循环迭代, 直到枚举完表中的所有元素, 循环结束

对于基于索引的表, 想要遍历所有的内容 pairs 和 ipairs 可以更方便

模式匹配

模式 描述
%a 字符
%c 控制字符
%d 数字
%l 小写字母
%p 标点符号
%s 空字符
%u 大写字符
%w 字母和数字
%x 十六进制
%z 表示为0

在C++中使用Lua库, 只需要三个头文件

  • lua.h : 用于核心功能, 这里的所有内容都有一个lua_前缀
  • luaxlib.h : 用于辅助库(auxlib). 辅助库提供了更多构建在lua.h核心函数之上的辅助函数. 这里的所有内容都有一个luaL_前缀
  • lualib.h 用于加载和构建lua库. 例如, luaL_openlibs函数打开所有标准库

Lua是用C实现的, 为了与C++一起工作, Lua提供了一个方便的包装器 lua.hpp

大多数Lua库函数如果成功都会返回0, 可以显示地将返回值与 LUA_OK 进行比较, 该值定义为0.

chunk : Lua中的编译单元称为chunk. 从语法上讲, chunk只是一个代码块. 当放入栈时, chunk是函数类型的值

stack : 每个Lua状态都在内部维护着一个stack. 栈中的每个元素都可以保存对Lua数据的引用

lua_pcall : 执行编译后的代码

Lua中的栈用于和C/C++之间的通信, C++代码和Lua代码都可以显示或隐式地对栈进行操作

Lua将不同类型的值压入栈中的方法:
void lua_pushnil (lua_State* L);
void lua_pushboolean (lua_State* L, int bool);
void lua_pushnumber (lua_State* L, lua_Number n);
void lua_pushinteger (lua_State* L, lua_Integer n);
void lua_pushstring (lua_State* L, const char* s);

上述方法的值取决于Lua库的配置方式以及操作系统的默认设置
大多数情况下, 将lua_Number映射为double, 将lua_Integer映射为long可能就足够了

lua_gettop()检查堆栈中有多少元素
两种访问栈元素的方式

使用lua_toX查询元素
int lua_toboolean (lua_State* L, int index);
int lua_tostring (lua_State* L, int index);
int lua_tonumber (lua_State* L, int index);
int lua_tointeger (lua_State* L, int index);

删除栈中前n个元素
void lua_pop (lua_State* L, int n);

C++和Lua之间的通信全都需要使用Lua栈
