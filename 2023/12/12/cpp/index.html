异常处理

异常处理的基本思想是将错误检测(在被调用函数中完成)与错误处理(在调用函数中完成)分开

位运算符

位运算仅适用于整数类型. 所有二元按位运算符都可以构成按位赋值运算符, 例如 &=, |=, ^=, <<=, >>=

按位运算符比算术运算符效率更高

&(按位与) : The AND of two corresponding bits yields a 1 if both bits are 1.
|(按位或) : The OR of two corresponding bits yields a 1 if either bit is 1.
^(按位异或) : The XOR of two corresponding bits yields a 1 only if two bits are different.
~(反码) : The operator toggles each bit from 0 to 1 and from 1 to 0.
<<(左移, 乘以2) : The operator shifts bits in the first operand left by the number of bits specified in the second operand, filling with 0s on the right.
>>(右移, 除以2) : The operator shifts bit in the first operand right by the number of bits specified in the second operand, filling with the highest (sign) bit on the left.

命令行参数

从命令行将参数传递给C++程序的main函数

int main(int argc, char* argv[])

argv(C字符串数组): 指定参数
argc: 指定参数的数量(程序名称也算作参数)

枚举类型

在枚举类型中的每个值称为枚举数, 是一个标识符, 而不是字符串
枚举类型的命名方式是每个单词的首字母大写, 而枚举数的命名方式类似于常量命名
